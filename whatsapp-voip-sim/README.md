# WhatsApp Call Deepfake Detection (Simulation)

## Setup
1. Install VB-Audio Virtual Cable
2. WhatsApp Speaker → CABLE Input
3. Windows Sound Settings:
   - Recording → CABLE Output
   - Enable "Listen to this device"
4. Run:
   streamlit run app.py

## Demo
- Make WhatsApp call
- App listens live
- Shows REAL / FAKE status
